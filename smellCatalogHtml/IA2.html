Incomplete Abstraction

This smell arises when an abstraction does not support complementary or interrelated methods completely.

Related smells: Incomplete Library Class

Reference

Girish Suryanarayana, Ganesh Samarthyam, and Tushar Sharma, Refactoring for Software Design Smells: Managing Technical Debt, 1st ed. Morgan Kaufmann, 2014.
