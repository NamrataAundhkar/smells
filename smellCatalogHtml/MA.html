Missing Abstraction

This smell arises when clumps of data or encoded strings are used instead of creating a class or an interface.

Related smells: Primitive Obsession Data Clumps

Reference

Girish Suryanarayana, Ganesh Samarthyam, and Tushar Sharma, Refactoring for Software Design Smells: Managing Technical Debt, 1st ed. Morgan Kaufmann, 2014.
