Spaghetti Code

This smell refers to an unmaintainable, incomprehensible code without any structure. Spaghetti Code does not exploit and prevents the use of object-orientation mechanisms and concepts.

Related smells:

Reference

W. Brown, R. Malveau, H. McCormick, and T. Mowbray, AntiPatterns: Refactoring software, architectures, and projects in crisis. John Wiley & Sons, 1998.
