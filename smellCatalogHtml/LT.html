Lazy Test

This occurs when several test methods check the same method using the same fixture (but for example check the values of different instance variables).

Related smells:

Reference

Arie Deursen, Leon M.F. Moonen, A. Bergh, and Gerard Kok, “Refactoring Test Code”, Technical Report, CWI, Amsterdam, The Netherlands, 2001
